Show 3 similar posts by category

== Description ==
A plugin to show the latest 3 posts by category

== Installation ==

1. Upload the `similar-3` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer
3. Go to your article page and the "See related articles" link will be at the bottom of your article, click on it and you will see the latest related articles.
